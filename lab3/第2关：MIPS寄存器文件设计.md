#### 电路引脚

| 信号 | 输入/输出 | 位宽 | 功能描述                                                |
| ---- | --------- | ---- | ------------------------------------------------------- |
| R1#  | 输入      | 5    | 第 1 个读寄存器的编号                                   |
| R2#  | 输入      | 5    | 第 2 个读寄存器的编号                                   |
| W#   | 输入      | 5    | 写入寄存器编号                                          |
| Din  | 输入      | 32   | 写入数据                                                |
| WE   | 输入      | 1    | 写使能信号，为 1 时在 CLK 上跳沿将 Din 数据写入W#寄存器 |
| CLK  | 输入      | 1    | 时钟信号，上跳沿有效                                    |
| RD1  | 输出      | 32   | R1# 寄存器的值，MIPS 寄存器文件中 0 号寄存器的值恒零    |
| RD2  | 输出      | 32   | R2# 寄存器的值，MIPS 寄存器文件中 0 号寄存器的值恒零    |
